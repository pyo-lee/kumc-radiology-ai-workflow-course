# KUMC Radiology AI Workflow Course

This repository provides course materials for radiology workflow enhancement using Generative AI, Agentic AI, and Colab-based hands-on practice.

# 의료 CSV 데이터 전처리 실습

본 강의는 Google Colab을 이용하여 의료 EMR-like CSV 데이터를 불러오고,  
결측치 확인, 변수 유형 구분, 기초 통계 계산, 간단한 AI 모델링 흐름을 실습하는 자료입니다.

## 사용 데이터셋

- Diabetes 130-US Hospitals for Years 1999-2008
- UCI Machine Learning Repository
- 목표: 퇴원 후 30일 이내 재입원 여부 예측

## Colab에서 바로 열기

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/pyo_lee/kumc-radiology-ai-workflow-course/blob/main/notebooks/[강의2] Python을 활용한 CSV 파일 구조 이해 및 기초 처리.ipynb)

## 실습 순서

1. CSV 파일 이해
2. 데이터셋 소개
3. Colab에서 데이터 불러오기
4. 데이터 구조 확인
5. 결측치 확인
6. 범주형 변수와 수치형 변수 구분
7. 기초 통계 및 시각화
8. AI 모델 학습 흐름 소개

## 실습 방법

1. 위의 `Open in Colab` 버튼을 클릭합니다.
2. Colab에서 노트북이 열리면 `파일 > 드라이브에 사본 저장`을 클릭합니다.
3. 코드 셀을 위에서부터 순서대로 실행합니다.
